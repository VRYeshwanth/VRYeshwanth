# 🚀 Featured Projects

## 🏆 Kaggle Dataset Evaluator Agent

An AI-powered application built for the **Google × Kaggle AI Agents Capstone Competition** that helps users quickly understand the quality and characteristics of a dataset before starting a Machine Learning project. The system combines automated data analysis with an AI agent to generate insights and recommendations through an interactive interface.

**Tech Stack:** Python • Pandas • Streamlit • FastMCP • Google ADK • Gemini API

### Features

- 📊 Dataset Analysis
- 📈 Dataset Quality Evaluation
- 🤖 ML Task Inference
- 💡 Project Recommendations
- 📝 Automated Dataset Reports
- ⚡ Google ADK + MCP Integration
- 🌐 Streamlit Frontend

## ❤️ Heart Disease Prediction

An end-to-end Machine Learning application that predicts the likelihood of heart disease using patient health information. The project follows the complete ML lifecycle from data preprocessing to deployment with an interactive web interface.

**Tech Stack:** Python • Scikit-learn • Pandas • NumPy • Matplotlib • Streamlit

### Features

- Exploratory Data Analysis
- Feature Engineering
- Model Comparison
- Hyperparameter Tuning
- Streamlit Deployment
